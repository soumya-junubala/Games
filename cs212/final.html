Lights Out

Explanation:

"Lights Out" is the game that is created with the specified code. Clicking on each cell to turn it off is the goal. Clicking on a cell flips it along with its neighboring cells. Until every cell is turned off, the game goes on.

The appearance, color scheme, and arrangement of the game elements are specified by the CSS styles. A grid of cells is used to symbolize the game board. A cell's color can be toggled by clicking on it, and the restart button restarts the game by creating a randomly chosen new board.

The logic of the game is controlled by JavaScript functions, which also generate a randomly solvable board, determine whether the game has been solved, toggle cells, render the board, and resume the game.

All things considered, the interactivity of the Lights Out game with its tidy and eye-catching design is made possible by the combination of HTML, CSS, and JavaScript.
